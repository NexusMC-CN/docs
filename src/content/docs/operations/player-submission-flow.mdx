## 适用范围

这篇文档适用于“找服玩”服务器投稿。它关注服务器信息是否真实、可连接、分类准确，并帮助玩家判断服务器是否适合加入。

::: note
找服玩投稿展示的是服务器信息，不是广告位。标题、截图和正文都应能被实际连接结果验证。
:::

## 状态流转

| 状态 | 说明 |
| --- | --- |
| `draft` | 草稿，仅作者可见。 |
| `pending` | 已提交审核。 |
| `approved` | 审核通过并公开展示。 |
| `rejected` | 审核未通过，需要修改后重提。 |

## 标准流程

1. 用户填写服务器标题、简介、正文、分类和连接地址。
2. 填写平台、版本、网络环境、正版验证状态等信息。
3. 服务端校验服务器地址和端口。
4. 用户保存草稿或提交审核。
5. 正式提交时检查正文长度、增强认证、内容限制和审核规则。
6. 审核通过后公开展示。

## 必填信息

| 字段 | 说明 |
| --- | --- |
| 标题 | 服务器名称或清晰标题。 |
| 简介 | 一句话说明服务器特色。 |
| 正文 | 完整介绍服务器玩法、规则、加入方式。 |
| 分类 | 生存、小游戏、综合等分类。 |
| 服务器地址 | 域名或 IP。 |
| 正版验证 | 明确是否开启在线正版验证。 |
| 网络环境 | 至少选择一个适用网络环境。 |

## 推荐补充信息

| 字段 | 说明 |
| --- | --- |
| 平台 | Java、基岩版或其他。 |
| 支持版本 | 让玩家判断客户端兼容性。 |
| 端口 | 默认 `25565`，非默认端口必须写清楚。 |
| 标签 | 玩法、特色、活动等关键词。 |
| 官网 | 服务器官网或介绍页。 |
| QQ 群 / Discord | 玩家交流入口。 |
| 封面和图集 | 展示出生点、建筑、玩法或活动截图。 |
| 在线人数和最大人数 | 可选的运营状态信息。 |

## 地址校验

服务器地址必须是可用于连接的目标。提交时会校验地址和端口，避免无效地址、危险网络目标或明显错误配置。

常见不合格写法：

- 空地址。
- 端口不是有效数字。
- 把网页地址当作服务器地址。
- 使用无法公开连接的内网地址。

如果有官网，请填到 `website`，不要填到服务器地址字段。

::: warning
不要把官网、宣传页、短链或群链接填进服务器地址字段。服务器地址应是玩家客户端可以连接的域名或 IP。
:::

## 草稿与正式提交

草稿阶段可以先保存不完整的介绍，但仍需要基础字段合理。

正式提交会额外检查：

- 服务器介绍正文长度。
- 地址和端口是否有效。
- 网络环境是否至少选择一项。
- 是否满足账号安全策略。
- 是否触发内容发布限制。

## 编辑规则

作者可以编辑自己的找服玩投稿。

编辑已公开或已驳回投稿时，普通用户通常会重新进入 `pending`。这些变更尤其需要重新审核：

- 服务器地址或端口变化。
- 标题、正文、分类大幅变化。
- 封面、图集或外部链接变化。
- 正版验证、网络环境、版本范围变化。

::: important
服务器地址、正版验证和网络环境属于核心信息。更新这些字段后，审核人员需要重新确认玩家是否仍能正常加入。
:::

## 常见失败原因

::: details 展开查看找服玩投稿失败原因

| 场景 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 标题不能为空 | 标题为空或只有空格。 | 填写服务器名称或明确标题。 |
| 描述不能为空 | 简介为空。 | 用一句话概括服务器特色。 |
| 请选择分类 | 分类为空或无效。 | 选择正确分类。 |
| 服务器地址不能为空 | `ip` 为空。 | 填写服务器域名或 IP。 |
| 请选择是否开启正版验证 | `onlineMode` 不是布尔值。 | 明确传 `true` 或 `false`。 |
| 请至少选择一个网络环境 | `networkEnvironments` 为空。 | 至少选择一个网络环境。 |
| 服务器地址无效 | 地址或端口校验失败。 | 检查域名、IP 和端口。 |
| 个人 Token 只能编辑自己的投稿 | 目标投稿作者不是当前用户。 | 使用作者账号操作。 |

:::

## 投稿前检查清单

::: details 展开检查清单

- 服务器地址能正常连接。
- 版本和平台写清楚。
- 正版验证状态真实。
- 正文包含玩法、规则、加入方式和联系方式。
- 截图来自真实服务器。
- 不夸大在线人数、福利或稳定性。

:::
