## 适用范围

这篇文档适用于站内视频投稿。视频投稿保存的是外部平台视频信息，当前不负责上传和托管视频原文件。

::: note
视频投稿记录的是外部视频页面和站内展示信息。原视频是否可播放，仍取决于外部平台。
:::

当前支持的视频来源：

- Bilibili
- 腾讯视频
- 抖音

## 当前流程

1. 用户提交视频链接和基础信息。
2. 服务端解析 `sourceUrl`，识别平台、视频 ID 和嵌入地址。
3. 服务端尝试抓取标题和封面等元数据。
4. 若未手动填写标题，则使用平台标题或兜底标题。
5. 若未手动填写封面，则优先使用平台元数据封面。
6. 视频创建后进入 `pending`，等待审核。
7. 审核通过后以 `approved` 状态公开展示；未通过则进入 `rejected`。

视频投稿当前没有草稿状态。创建成功后直接进入待审核。

::: important
视频投稿创建成功后会直接进入 `pending`，当前没有 `draft` 草稿状态。
:::

## 请求字段

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `sourceUrl` | 是 | 视频链接。必须能被解析为 Bilibili、腾讯视频或抖音视频。 |
| `categoryId` | 否 | 视频分区。不传时使用默认分区。 |
| `title` | 否 | 标题。不传时尝试从视频平台元数据读取。 |
| `description` | 否 | 视频简介。建议补充视频看点、版本或适用场景。 |
| `coverImage` | 否 | 手动封面。填写后优先于平台抓取封面。 |
| `tags` | 否 | 标签。建议 2 到 5 个即可。 |
| `customSlug` | 否 | 自定义公开路径。 |
| `organizationId` | 否 | 个人 API Token 暂不支持组织身份。 |

## 链接解析规则

视频链接必须是真实、可识别的视频页面地址。不要提交短链跳转、聚合页、合集页、用户主页或搜索结果页。

::: warning
短链、合集页、直播间、用户主页和搜索结果页通常无法稳定解析。请尽量使用原始视频详情页 URL。
:::

解析失败时，服务端会返回：

```json
{
  "error": "仅支持 Bilibili、腾讯视频或抖音链接"
}
```

处理方式：

- 换成原始视频页面 URL。
- 确认视频仍可公开访问。
- 避免带过多追踪参数。
- 确认不是直播间、合集页或用户主页。

## 封面与标题

标题和封面可以由平台元数据自动补全，但建议人工确认：

- 平台标题过长时应手动精简。
- 平台封面模糊、低清或不准确时应手动上传封面。
- 封面必须和视频内容一致，不能用无关图吸引点击。

::: caution
误导性封面、标题党或与视频内容无关的标签，可能导致视频被拒绝或后续下架。
:::

如果更新时重新传入 `sourceUrl`，服务端会重新解析视频来源，并可能重新获取平台封面。已手动填写的封面仍应作为更高优先级处理。

## 状态流转

| 状态 | 说明 |
| --- | --- |
| `pending` | 创建或更新后等待审核。 |
| `approved` | 审核通过并公开展示。 |
| `rejected` | 审核未通过，需要修改后重新提交。 |

更新已公开视频时，标题、简介、分区、封面、标签或源链接变化都可能使视频重新进入 `pending`。

## 审核重点

视频审核主要关注：

- 链接是否有效。
- 视频内容是否与标题、简介、封面一致。
- 是否存在搬运争议或授权问题。
- 分区是否正确。
- 是否包含违规、诱导、欺诈或与站点无关内容。

## 常见失败原因

::: details 展开查看视频投稿失败原因

| 场景 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 链接无法解析 | 平台不支持、链接不是视频页、视频不可访问。 | 换成支持平台的公开视频链接。 |
| 视频分区不存在 | `categoryId` 无效。 | 选择当前站点存在的视频分区，或不传让系统使用默认分区。 |
| 被拒绝 | 内容与站点主题不符、封面误导、版权风险。 | 根据驳回原因修改标题、简介、封面或来源。 |
| 更新时报无权编辑 | 当前用户不是视频作者，也没有可用管理权限。 | 使用作者账号或对应内容所属账号操作。 |
| 个人 Token 不能组织投稿 | 请求中传了非空 `organizationId`。 | 移除 `organizationId`。 |

:::

## 投稿前检查清单

::: details 展开检查清单

- 链接能在无登录状态下打开。
- 标题能准确描述视频内容。
- 简介说明视频看点、适用版本或相关资源。
- 封面清晰且不误导。
- 分区和标签准确。
- 确认视频来源和授权没有争议。

:::
