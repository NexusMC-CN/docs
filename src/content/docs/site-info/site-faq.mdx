## 账号与登录

::: details 忘记密码怎么办

如果账号绑定了邮箱，可以通过重置密码流程找回。若启用了两步验证或通行密钥，请按页面提示完成额外验证。

如果邮箱不可用，建议联系管理员，并准备能证明账号归属的信息。

:::

::: details 可以修改用户名吗

是否允许修改用户名取决于当前站点配置。若页面没有提供修改入口，说明该能力可能暂未开放，或需要管理员协助。

:::

::: details 为什么需要绑定邮箱或手机号

绑定信息用于账号找回、安全验证、通知发送和风控判断。不同登录方式下，可用的绑定项可能不同。

:::

## 投稿与审核

::: details 为什么我的内容需要审核

帖子、资源、找服玩服务器和视频投稿都可能进入审核队列。审核用于确认内容是否符合站点规范、分类是否正确、链接是否安全，以及是否存在广告、侵权或违规信息。

:::

::: details 草稿和待审核有什么区别

| 状态 | 含义 |
| --- | --- |
| `draft` | 草稿，仅作者可见，通常不会进入公开列表。 |
| `pending` | 已提交审核，等待版主或管理员处理。 |
| `approved` | 审核通过，公开展示。 |
| `rejected` | 审核未通过，需要修改后重新提交。 |

:::

::: details 已公开内容再次编辑后为什么又进入审核

对已公开内容进行关键字段更新时，系统可能重新进入审核流程。常见触发项包括标题、正文、下载链接、分类、封面、视频链接等。

:::

## 内容与互动

::: details 点赞、收藏、评论会公开吗

点赞、收藏和评论的展示方式取决于具体页面。评论和回复通常是公开互动；收藏可能只在个人空间或自己的列表中展示。

:::

::: details 可以删除自己的内容吗

不同内容类型的删除能力可能不同。部分内容只允许编辑或设为草稿，不能直接删除；涉及审核记录、公开讨论或安全记录的内容，可能需要管理员处理。

:::

::: details 为什么看不到某篇帖子

可能原因包括：

- 内容还在草稿或审核中。
- 内容被作者设置了可见性或访问密码。
- 内容被锁定、隐藏或移入处理流程。
- 你没有对应板块或内容的访问权限。

:::

## 私信与通知

::: details 为什么收不到通知

请检查：

- 账号通知开关是否关闭。
- 邮箱通知是否启用并完成验证。
- 浏览器推送权限是否被拒绝。
- 内容是否实际触发了通知事件。

:::

::: details 私信可以被举报吗

如果私信存在骚扰、诈骗、广告导流、恶意链接或其他违规内容，应通过站点提供的举报入口提交。管理员会根据记录处理。

:::

## 搜索与统计

::: details 搜索结果为什么不完整

搜索通常只展示你有权限查看的公开内容。草稿、待审核、隐藏、私密或受限内容不会正常出现在公开搜索结果中。

:::

::: details 首页统计为什么会延迟

首页社区统计存在缓存。资源总数、帖子总数、今日新发和在线人数不一定实时刷新，短时间内存在延迟属于正常情况。

:::

## 安全与违规

::: details 被封禁后怎么办

先查看站内通知或提示中的封禁原因。如果认为处理有误，可以按站点申诉渠道联系管理员。申诉时请说明账号、涉及内容、时间和你的解释。

:::

::: details 发现违规内容怎么办

优先使用举报功能。举报时尽量提供明确原因，不要在评论区继续争吵或扩大冲突。

:::

::: details 什么内容容易被拒绝

常见原因包括：

- 标题或正文过短，无法说明内容。
- 下载链接不可访问或存在风险。
- 分类选择错误。
- 未说明转载、授权或依赖关系。
- 广告、引流、诈骗、恶意链接或侵犯他人权益。

:::
